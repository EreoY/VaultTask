8. Rules (ห้ามทำ)

1. ห้าม hardcode สี — ทุกสีต้องผ่าน CSS Variable เท่านั้น (var(--xxx)) ไม่งั้น Light Mode จะพัง

2. ห้ามใช้ #000000 หรือ #FFFFFF — ใช้ var(--bg-deep) และ var(--text-primary) แทน

3. ห้ามใช้ bg-black หรือ bg-white — แม้ overlay ก็ต้องใช้ color-mix กับ variables

4. ห้ามลืม transition — ทุก element ที่เปลี่ยนสีต้องมี transition: all 0.3s

5. ฟอนต์หัวข้อต้องเป็น Playfair Display — ห้ามใช้ Inter กับหัวข้อใหญ่/ชื่อบอท

6. ฟอนต์เนื้อหาต้องเป็น Inter — ห้ามใช้ Serif กับข้อความที่อ่านบ่อยๆ
